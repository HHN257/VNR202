# Quick Start Guide

## Get Started Fast

1. Install dependencies
```bash
npm install
```
2. Run dev server
```bash
npm run dev
```
3. Open http://localhost:3000 to view the site.

## What you get
- Trang chủ giới thiệu nghị quyết và điều hướng tới nội dung đầy đủ.
- Trang `/presentation` với đủ 7 phần nội dung nghị quyết Trung ương 5 khóa VIII.
- Mục `#tomtat` để xem tóm tắt nhanh.

## Notes
- Không cần Supabase hay biến môi trường.
- Minigame và leaderboard đã được gỡ bỏ; đây là phiên bản tập trung vào nội dung văn bản.
